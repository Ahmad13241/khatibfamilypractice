Forms Directory for Khatib Family Practice
=====================================

This directory should contain the following fillable PDF forms:

1. financial-policy.pdf - Contains the practice's financial policies and payment information
2. records-request.pdf - Form for patients to request medical records
3. office-policies.pdf - General office policies and procedures

These files should be fillable PDF forms that patients can download, complete electronically, and bring to their appointment or submit electronically.

The HTML files in the website reference these PDFs using the following paths:
- ../forms/financial-policy.pdf
- ../forms/records-request.pdf
- ../forms/office-policies.pdf

Please ensure these exact filenames are used when adding the actual PDF forms.
